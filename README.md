# Movie-db
此為本人利用Vue串接國外電影資料庫API所做的project  
[demo](https://a8692003.github.io/movie-db/)  
# 簡介  
主要功能以下  
1.有查看目前熱門電影，最新上映，即將上映  
2.點擊電影內部細節會有電影的分類、劇情、演員卡司、劇照、海報、預告片、推薦類似電影  
3.可以透過搜尋功能查看想查看的電影，也透過旁邊的按鈕來查看戲劇或電影  
4.將各戲劇電影依照類型作區分，喜愛動作片可以點進此分類查看所有動作片

# 使用技術
1.Vue  
2.Vuex  
3.Vue Router   
4.HTML  
5.CSS/SCSS  
6.Bootstrap  
7.Swiper
